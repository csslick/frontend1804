  1. 로고디자인 과제(제출형)

    - 주제: 개인 브랜드 또는 미래의 내 회사의 브랜드

    - 브랜드의 의도(컨셉)와 룩엔필에 대해 소개하시오. ppt(pdf)로 별첨

    1. 브랜드 아이덴티티
    2. 색상
    3. 타이포그래피
    4. 문양

    - 시안을 2가지 형태로 제시(A, B)

    - 발표 후 시안에 대한 호응도

    <예시>

    • 배경색: 말 그대로 배경에 사용되는 바탕색
    • 메인컬러(Main color): 주인공이 되는 색: 웹 사이트의 내용과 성격을 분명하게 보여주는 역할
    • 서브컬러(Sub color): 메인컬러를 도와주는 역할. 메인컬러와 어우러져서 전체 이미지를 만들어냄
    • 액센트 컬러(Accent color): 배색에 포인트, 강조점을 주는 색. 보통 작은 면적에 강하게 삽입

  2. 이미지를 활용하여 외곽을 패스 도형으로 만들어 다음과 같은 타이포그래피를 제작하시오.
